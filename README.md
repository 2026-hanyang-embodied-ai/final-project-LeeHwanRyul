# Policy Sharing and Agent Identity in Multi-Agent Reinforcement Learning

This project studies how policy parameter sharing affects cooperative multi-agent reinforcement learning under partial observability.

Three IPPO variants are compared in the MPE Simple Spread environment:

* Shared policy
* Shared policy with agent identity
* Independent policies

A random policy is also evaluated as a baseline.

* `results/`: CSV files generated by the notebook

## Dataset

This project does not use an external dataset. Training samples are generated online through interaction with the MPE2 Simple Spread simulation. The evaluation seeds and experiment configuration are included in `final-project.ipynb`.

## Running the Project

1. Open `final-project.ipynb`.
2. Set `FAST_RUN = True` to test the code.
3. Restart the kernel and run all cells.
4. Set `FAST_RUN = False` for the final experiment.
5. Restart the kernel and run all cells again.

The final experiment trains three policy variants using three random seeds.
